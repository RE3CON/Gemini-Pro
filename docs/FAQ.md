# Frequently Asked Questions ❓

**Q: Does this work on mobile browsers?**
A: Yes, if you use a mobile browser that supports UserScripts (like Kiwi Browser or Firefox Nightly with extensions).

**Q: Will this break websites?**
A: Some highly secure banking or streaming sites might detect the canvas poisoning. You can easily toggle the script off in your UserScript manager for those specific sites.

**Q: How do I report a bug?**
A: Please use the [Issues](https://github.com/RE3CON/Gemini-AI/issues) tab on our GitHub repository. For security vulnerabilities, refer to our `SECURITY.md` policy.
